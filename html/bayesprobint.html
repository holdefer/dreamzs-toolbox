bayesprobint
Syntax 
Y = bayesprobint(X,arg2,modeStr)
Description

Y = bayesprobint(X,arg2,modeStr)

Calculates Bayesian probability intervals. Variable X should be size NxM, with N rows of model results, each row having M elements. Variable modeStr can be 'vec' for output as vector data, or 'mat' for output as 2-D matrix. If 'vec' is selected, input variable arg2 is interpreted as percentiles. Function prctile from the statistics toolbox is required in this case. If 'mat' is selected, input variable arg2 is interpreted as the bin boundaries of the 2-D matrix.
