continuedream
Syntax 
runCondition = continuedream(dreamPar,nModelEvals)
Description
This function determines whether the optimization procedure should continue. Termination is based on one of the following:
  1. the maximum number of iterations as specified by dreamPar.nModelEvalsMax has been reached;
  2. the optimization end time as specified by dreamPar.optEndTime has been exceeded;
  3. convergence has been achieved.
