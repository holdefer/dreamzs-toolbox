Toolbox contents
generateCrossoverValues
Syntax 
allCrossoverValues = generateCrossoverValues(dreamPar,pCrossover)
Input arguments
  • dreamPar is a structure containing the parameters of the DREAM algorithm.
  • pCrossover is a vector with dreamPar.nCrossoverValues elements containing the current crossover probabilities.
Output arguments
  • allCrossoverValues is a dreamPar.nSeq x dreamPar.nOffspringPerSeq matrix that contains the crossover probabilities.
Description
Associates to each point in a generation (in all the sequences) a crossover probability (which is one of 1/nCrossoverValues, 2/nCrossoverValues, ... ,1 ). The distribution of the probability values is based on a multinomial distribution.
