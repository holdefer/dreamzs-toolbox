getDiffEvolStrategy
Syntax 
diffEvolStrategy = getDiffEvolStrategy(dreamPar)
Input arguments
  • dreamPar is a structure containing the parameters of the DREAM algorithm.
Output arguments
  • diffEvolStrategy contains the modified proposed points in each of the chains.
Description
Associates to each sequence a differential evolution strategy, which is basically a number between 1 and dreamPar.nDiffEvolPairs, representing the number of pairs used in the generateOffspring function.
