prepseqarray
Syntax 
sequences = prepseqarray(dreamPar,sequences)
Description
Function prepseqarray prepares the array sequences for the next generation by pre-allocation. This is achieved by concatenating the existing array sequences with an array of size [dreamPar.nOffspringPerCompl,size(sequences,2),dreamPar.nCompl], filled with NaNs.
