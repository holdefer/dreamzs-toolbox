Toolbox contents
reflectBounds
Syntax 
[propChild] = reflectBounds(propChild,dreamPar)
Input arguments
  • dreamPar is a structure containing the parameters of the DREAM algorithm.
  • propChild is a matrix with dreamPar.nSeq rows containing the proposed points in each of the chains.
Output arguments
  • propChild contains the modified proposed points in each of the chains.
Description
This function checks whether the proposed points in every chain are within bounds and, if not, reflects the point against the respective bound. If, after applying this step, the new point is still outside the boundaries, a random point within boundaries is generated.
