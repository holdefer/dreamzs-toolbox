stratranddraw
Syntax 
sRandDraw = stratranddraw(dreamPar)
Description

Stratified random sampling of points from the valid parameter space as defined by dreamPar.parRangeMin and dreamPar.parRangeMax.

stratranddraw divides each axis (1..dreamPar.nOptPars) into X segments, such that X is the largest positive integer for which X^dreamPar.nOptPars < dreamPar.nSamples. This way the parameter space is divided into nStratRandSamples hyperrectangles. From each hyperrectangle, exactly one random sample is taken. When the number of required samples dreamPar.nSamples is larger than nStratRandSamples, an additional nSamples-nStratRandSamples samples are taken using uniform random sampling of the parameter space (see unifranddraw).
