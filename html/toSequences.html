toSequences
Syntax 
sequences = toSequences(dreamPar,evalResults)
Description 
complexes = toSequences(dreamPar,evalResults)
Function toSequences partitions the rows from the variable evalResults over the dreamPar.nSeq Markov Chains (aka sequences). This is achieved by distributing evenly the points among the sequences.
