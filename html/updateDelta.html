updateDelta
Syntax
deltaTot = updateDelta(dreamPar,deltaTot,acceptedChild,lastPointsFromEverySeq, currentCrossoverValues)
Input arguments
  • dreamPar is a structure containing the parameters of the DREAM algorithm.
  • deltaTot is a vector of dreamPar.nCrossoverValues elements containing the the total normalized Euclidean distance between the old points and the new points (in the chains updated by that specific crossover value).
  • acceptedChild is a matrix with dreamPar.nSeq rows containing the accepted points in each of the chains.
  • lastPointsFromEverySeq is a matrix with dreamPar.nSeq rows containing the current points in each of the chains.
  • currentCrossoverValues is a dreamPar.nSeq column vector that contains the crossover probabilities for the specific offspring (1...dreamPar.nOffspringPerSeq).
Output arguments
  • deltaTot is the updated deltaTot taking into account the current generation.
Description
For each crossover value it calculates the total normalized Euclidean distance between the old points in lastPointsFromEverySeq and the new points in acceptedChild (in the chains updated by that specific crossover value)
