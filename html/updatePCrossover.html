updatePCrossover
Syntax
[pCrossover,crossoverCount] = updatePCrossover(dreamPar,allCrossoverValues,deltaTot,crossoverCount)
Input arguments
  • dreamPar is a structure containing the parameters of the DREAM algorithm.
  • allCrossoverValues is a dreamPar.nSeq x dreamPar.nOffspringPerSeq matrix that contains the crossover probabilities.
  • deltaTot is a vector of dreamPar.nCrossoverValues elements representing the averaged normalized jumping distance.
  • crossoverCount is a vector with dreamPar.nCrossoverValues elements containing the number of times a particular crossover value is used.
Output arguments
  • pCrossover is a vector with dreamPar.nCrossoverValues elements containing the current crossover probabilities updated taking into account the current iteration.
  • crossoverCount is a vector with dreamPar.nCrossoverValues elements containing the updated crossoverCount taking into account the current iteration.
Description
Updates the probabilities of the various crossover values.
