updateSequences
Syntax
[sequences, wasLastPointStored] = updateSequences(dreamPar, sequences, acceptedChild, iteration)
Input arguments
  • dreamPar is a structure containing the parameters of the DREAM algorithm.
  • sequences is a 3-dimensional array containing either all the points in each of the chains, if dreamPar.reducedSampleCollection = false, or every dreamPar.reducedSampleFrequencythe point in each sequence, if dreamPar.reducedSampleCollection = true.
  • acceptedChild is a matrix with dreamPar.nSeq rows containing the accepted points in each of the chains.
  • iteration is the current iteration number.
Output arguments
  • sequences is the updated sequences array.
  • wasLastPointStored is a boolean value that marks if the sequences array was updated (which is not always the case in the case of dreamPar.reducedSampleCollection = true).
Description
Updates the sequences array by adding the last accepted points in each of the sequences. Whether a point is added or not depends of the value of the dreamPar.reducedSampleCollection parameter.
